# 아하소풍 엔딩노트 WLC 2급 실습판 콘텐츠 구조 설계

## 1. 기획 방향

아하소풍 엔딩노트 WLC 2급 실습판은 WLC 2급 과정을 마친 수강생이 직접 쓰고, 가족과 나누고, 교육 현장에서 기초 실습 자료로 활용할 수 있도록 설계함.

이 자료는 죽음을 먼저 말하기보다 삶의 의미, 존엄, 가족 배려, 오늘의 실천을 중심으로 구성함. 법적·의료적 판단이 필요한 내용은 예비 기록으로만 다루며, 정식 작성은 관련 기관 또는 전문가 상담으로 연결함.

## 4. 장별 콘텐츠 설계

### 1장. 나의 삶을 바라보는 첫 기록

#### 해당 주차와의 연결

- 1주차 수업에서 웰다잉을 "삶을 잘 정리하고 오늘을 더 잘 사는 준비"로 이해함
- 첫 장은 수강생이 부담 없이 자기 삶을 바라보는 출발점임

#### 학습 목적

- 내 삶을 한 문장으로 정리함
- 엔딩노트를 무거운 문서가 아니라 나를 돌보는 기록으로 받아들임

#### 작성 질문

- 나는 지금까지 어떤 삶을 살아왔음?
- 내 삶에서 가장 고마웠던 순간은 언제였음?
- 나를 버티게 해 준 사람이나 힘은 무엇이었음?
- 앞으로 남은 시간에 꼭 지키고 싶은 태도는 무엇임?
- 오늘부터 내 삶을 위해 해 보고 싶은 작은 실천은 무엇임?

#### 수강생 작성란 예시

| 구분 | 나의 기록 |
| --- | --- |
| 내 삶을 표현하는 단어 3개 | 가족, 성실, 배움 |
| 가장 고마운 기억 | 어려울 때 가족이 함께해 준 일 |
| 나를 버티게 한 힘 | 다시 시작할 수 있다는 믿음 |
| 앞으로의 다짐 | 하루 한 번 고마운 사람에게 마음을 표현함 |

#### 가족과 공유할 문장

내 삶에는 부족함도 있었지만, 가족과 함께한 시간 덕분에 고마운 순간이 많았음.

#### 강사용 진행 메모

- 첫 시간에는 깊은 이야기를 강요하지 않음
- 단어 3개만 써도 충분하다고 안내함
- "죽음 준비"보다 "삶을 돌아보는 첫 기록"으로 설명함

### 2장. 내가 소중히 여기는 가치

#### 해당 주차와의 연결

- 2주차 수업에서 웰빙과 웰다잉의 공통 기준인 가치와 존엄을 다룸
- 이후 의료, 돌봄, 장례, 가족 대화의 기준이 되는 장임

#### 학습 목적

- 내가 중요하게 여기는 삶의 기준을 찾음
- 가족이 나의 선택을 이해할 수 있도록 가치 중심으로 표현함

#### 작성 질문

- 나는 어떤 삶을 좋은 삶이라고 생각함?
- 마지막까지 지키고 싶은 나의 존엄은 무엇임?
- 내가 가족에게 가장 바라는 배려는 무엇임?
- 내가 원하지 않는 삶의 모습은 무엇임?
- 중요한 결정을 할 때 가족이 기억해 주었으면 하는 기준은 무엇임?

#### 수강생 작성란 예시

| 가치 | 나에게 중요한 이유 | 가족에게 전할 말 |
| --- | --- | --- |
| 존엄 | 내가 나답게 선택하고 싶음 | 내 뜻을 먼저 물어봐 주면 좋겠음 |
| 평안 | 가족이 죄책감을 갖지 않기를 바람 | 서로 미안함보다 고마움을 기억했으면 함 |
| 배려 | 한 사람에게 부담이 몰리지 않기를 바람 | 함께 상의해서 결정해 주면 좋겠음 |

#### 가족과 공유할 문장

내가 바라는 선택의 기준은 오래 버티는 것만이 아니라, 나답게 존엄을 지키고 가족의 마음도 덜 무겁게 하는 것임.

#### 강사용 진행 메모

- 정답을 찾는 시간이 아니라 자기 기준을 찾는 시간임을 강조함
- "좋은 죽음"보다 "좋은 삶의 마무리"라는 표현을 사용함

### 3장. 나의 관계 지도와 감사 인사

#### 해당 주차와의 연결

- 3주차 수업에서 관계 회복, 감사, 미안함, 화해의 말을 다룸
- 아하소풍 엔딩노트의 관계 회복 편지 쓰기와 연결됨

#### 학습 목적

- 내 삶에 중요한 사람을 떠올림
- 감사와 미안함을 짧은 문장으로 표현함
- 가족 대화의 시작 문장을 준비함

#### 작성 질문

- 내 삶에서 꼭 고맙다고 말하고 싶은 사람은 누구임?
- 아직 미안하다고 말하지 못한 일이 있음?
- 다시 연락하거나 만나고 싶은 사람은 누구임?
- 내가 먼저 건넬 수 있는 짧은 말은 무엇임?
- 관계 회복을 위해 이번 주에 할 수 있는 작은 행동은 무엇임?

#### 수강생 작성란 예시

| 사람 | 전하고 싶은 마음 | 실천 방법 |
| --- | --- | --- |
| 배우자 | 함께 살아줘서 고마움 | 이번 주에 손편지 쓰기 |
| 자녀 | 늘 걱정만 해서 미안함 | 식사 자리에서 직접 말하기 |
| 친구 | 오래 연락하지 못해 아쉬움 | 안부 문자 보내기 |

#### 가족과 공유할 문장

나는 가족에게 완벽한 사람은 아니었지만, 함께한 시간을 고맙게 기억하고 있음.

#### 강사용 진행 메모

- 관계 이야기는 감정이 올라올 수 있으므로 충분한 침묵 시간을 줌
- 공개 발표보다 옆 사람과 한 문장만 나누는 방식이 안전함

### 4장. 가족에게 전하고 싶은 말

#### 해당 주차와의 연결

- 4주차 수업에서 가족 소통과 마음 전달을 실습함
- 3주차의 관계 정리를 가족 편지 초안으로 확장함

#### 학습 목적

- 가족에게 남기고 싶은 핵심 메시지를 정리함
- 고마움, 부탁, 응원을 짧고 쉬운 말로 표현함

#### 작성 질문

- 가족에게 가장 먼저 고맙다고 말하고 싶은 것은 무엇임?
- 가족에게 미안하다고 말하고 싶은 것은 무엇임?
- 가족이 앞으로 살아가며 기억해 주었으면 하는 말은 무엇임?
- 가족에게 부탁하고 싶은 생활상의 배려는 무엇임?
- 가족 각자에게 응원하고 싶은 말은 무엇임?

#### 수강생 작성란 예시

> 사랑하는 가족에게  
> 내가 바라는 것은 거창한 것이 아님. 서로 너무 미안해하지 말고, 각자의 삶을 건강하게 살아가면 좋겠음. 나와 함께한 시간을 고맙게 기억해 주면 충분함.

#### 가족과 공유할 문장

내가 가족에게 가장 바라는 것은 서로 죄책감보다 고마움을 기억하며 살아가는 것임.

#### 강사용 진행 메모

- "고마움 1문장, 부탁 1문장, 응원 1문장"으로 시작하게 함
- 가족 관계가 복잡한 수강생은 자기 자신에게 쓰는 편지로 바꾸어도 됨

### 5장. 나의 건강과 돌봄 희망

#### 해당 주차와의 연결

- 5주차 수업에서 건강, 돌봄, 가족 부담 줄이기를 다룸
- 2주차의 가치 기준을 실제 돌봄 희망으로 연결함

#### 학습 목적

- 아플 때 바라는 돌봄 환경과 도움 방식을 정리함
- 가족이 돌봄 결정을 할 때 참고할 생활 정보를 남김

#### 작성 질문

- 내가 아플 때 가장 중요하게 생각하는 것은 무엇임?
- 집, 병원, 요양시설 중 어떤 환경을 더 편안하게 느끼는지 적어 봄
- 가족이 나를 돌볼 때 꼭 알아야 할 습관은 무엇임?
- 내가 싫어하거나 불편해하는 돌봄 방식은 무엇임?
- 가족의 부담을 줄이기 위해 미리 상의하고 싶은 것은 무엇임?

#### 수강생 작성란 예시

| 항목 | 나의 희망 |
| --- | --- |
| 편안한 장소 | 가능하면 익숙한 공간에서 지내고 싶음 |
| 좋아하는 돌봄 방식 | 조용히 설명해 주고, 내 의견을 먼저 물어봐 주면 좋겠음 |
| 불편한 방식 | 내 앞에서 가족끼리 다투지 않았으면 함 |
| 가족에게 부탁 | 한 사람이 모든 부담을 지지 않았으면 함 |

#### 가족과 공유할 문장

내 돌봄은 가족 한 사람의 책임이 아니라, 함께 상의하고 나누어 결정해 주면 좋겠음.

#### 강사용 진행 메모

- 의료 판단을 대신하는 시간이 아니라 가족 대화의 출발점을 만드는 시간임을 설명함
- 돌봄 기관, 병원 상담, 공공기관 정보를 확인할 수 있음을 안내함

### 6장. 사전연명의료의향서 예비 작성

#### 해당 주차와의 연결

- 6주차 수업에서 사전연명의료의향서의 뜻과 작성 절차를 이해함
- 2주차 가치 기준과 5주차 돌봄 희망을 의료 결정 예비 생각으로 연결함

#### 학습 목적

- 사전연명의료의향서를 쉽게 이해함
- 정식 작성 전 나의 생각과 가족에게 전할 기준을 정리함

#### 작성 질문

- 나는 회복 가능성이 매우 낮은 상황에서 어떤 점을 가장 중요하게 생각함?
- 치료를 계속할지 판단할 때 고려하고 싶은 기준은 무엇임?
- 가족이 내 뜻을 설명해야 한다면 꼭 말해 주었으면 하는 내용은 무엇임?
- 가족이 이 문제로 죄책감을 갖지 않도록 전하고 싶은 말은 무엇임?
- 정식 상담 전에 더 알고 싶은 질문은 무엇임?

#### 수강생 작성란 예시

| 질문 | 나의 예비 생각 |
| --- | --- |
| 치료 결정 기준 | 회복 가능성과 고통 정도를 함께 고려하고 싶음 |
| 가족에게 부탁 | 내 뜻을 두고 가족끼리 다투지 않았으면 함 |
| 상담 전 질문 | 정식 등록 절차와 변경 방법을 알고 싶음 |

#### 가족과 공유할 문장

내 의료 결정은 가족을 힘들게 하려는 뜻이 아니라, 미리 상의해서 서로의 부담을 줄이기 위한 준비임.

#### 강사용 진행 메모

- 반드시 안내 문구를 읽어 줌: “이 양식은 법적 효력이 없으며, 정식 작성은 관련 기관 또는 전문가 상담이 필요함.”
- 의료적 선택의 정답을 제시하지 않음
- 가족과 대화할 질문을 찾는 데 집중함

### 7장. 나의 장례 희망

#### 해당 주차와의 연결

- 7주차 수업에서 장례 문화와 가족 배려를 다룸
- 4주차 가족 메시지와 5주차 가족 부담 줄이기를 장례 희망으로 연결함

#### 학습 목적

- 장례 방식, 분위기, 비용, 연락 범위를 미리 생각함
- 가족이 장례를 준비할 때 겪는 혼란과 부담을 줄임

#### 작성 질문

- 나는 어떤 분위기의 장례를 원함?
- 꼭 연락했으면 하는 사람은 누구임?
- 가족이 장례 비용을 정할 때 지켜주었으면 하는 기준은 무엇임?
- 장례식에서 사용했으면 하는 사진, 음악, 문구가 있음?
- 가족이 장례 후 나를 어떻게 기억해 주면 좋겠음?

#### 수강생 작성란 예시

| 항목 | 나의 희망 |
| --- | --- |
| 장례 분위기 | 차분하고 따뜻한 인사를 나누는 자리 |
| 연락 대상 | 가까운 가족, 친한 지인, 동료 일부 |
| 비용 원칙 | 과한 지출보다 가족이 부담되지 않는 범위 |
| 음악/사진/문구 | 밝은 사진과 내가 좋아하던 음악 |

#### 가족과 공유할 문장

내 장례는 보여주기 위해 애쓰는 자리가 아니라, 서로 고마움을 나누고 마음을 정리하는 시간이었으면 함.

#### 강사용 진행 메모

- 반드시 안내 문구를 넣음: “이 양식은 법적 효력이 없으며, 정식 작성은 관련 기관 또는 전문가 상담이 필요함.”
- 장례 방식의 옳고 그름을 평가하지 않음
- 가족 부담을 줄이는 관점으로 진행함

### 8장. 생전 장례식과 마지막 인사

#### 해당 주차와의 연결

- 8주차 수업에서 생전 장례식, 감사 모임, 마지막 인사 준비를 다룸
- 3주차 관계 지도와 4주차 가족 편지를 살아 있을 때 전하는 활동으로 확장함

#### 학습 목적

- 살아 있을 때 전하고 싶은 감사와 화해의 말을 준비함
- 생전 장례식을 삶을 축하하고 마음을 나누는 자리로 이해함

#### 작성 질문

- 살아 있을 때 꼭 만나고 싶은 사람은 누구임?
- 그 사람에게 전하고 싶은 감사의 말은 무엇임?
- 함께 보고 싶은 사진이나 들려주고 싶은 음악은 무엇임?
- 작은 감사 모임을 연다면 어떤 분위기였으면 좋겠음?
- 지금 바로 전할 수 있는 한 문장은 무엇임?

#### 수강생 작성란 예시

| 항목 | 나의 계획 |
| --- | --- |
| 초대하고 싶은 사람 | 가족, 오랜 친구 3명 |
| 나누고 싶은 이야기 | 함께한 시간에 대한 감사 |
| 함께하고 싶은 활동 | 사진 보기, 짧은 편지 읽기, 좋아하는 노래 듣기 |

#### 가족과 공유할 문장

나는 이별을 기다리기보다, 살아 있는 동안 서로 고맙다고 말하는 시간을 갖고 싶음.

#### 강사용 진행 메모

- "생전 장례식"이 부담스러우면 "삶 감사 모임"으로 설명함
- 실제 행사를 강요하지 않고, 편지나 가족 식사처럼 작게 실천할 수 있음을 안내함

### 9장. 유언과 재산 정리 예비 점검

#### 해당 주차와의 연결

- 7주차 장례 희망과 함께 가족 갈등을 줄이는 실제 준비로 연결함

#### 학습 목적

- 유언의 다섯 가지 방식을 이해함
- 정식 작성이 필요한 이유를 알게 함

#### 작성 질문

- 내가 정리해 두어야 할 재산, 물건, 계정은 무엇임?
- 가족이 꼭 알아야 할 보관 장소와 연락처는 무엇임?
- 가족에게 물려주고 싶은 물건이나 기록은 무엇임?
- 재산보다 더 중요하게 전하고 싶은 뜻은 무엇임?
- 유언장 정식 작성을 위해 상담이 필요한 부분은 무엇임?

#### 수강생 작성란 예시

| 구분 | 예비 점검 내용 | 확인 필요 |

| 통장/보험 | 주요 금융기관 목록 작성 | 가족에게 보관 위치 공유 |
| 중요한 물건 | 사진, 반지, 문서 | 누구에게 전할지 정리 |

내가 미리 정리하는 이유는 재산보다 가족의 평안을 지키고, 불필요한 다툼을 줄이기 위해서임.

#### 강사용 진행 메모

- 본 장에서는 "정식 요건 확인이 중요함"을 강조함

### 10장. 나의 묘비명과 기억될 문장

## 6. 향후 확장 설계

### 수강생용 작성 워크북 PDF

- 각 장을 2쪽 단위로 구성함
- 왼쪽은 쉬운 설명과 예시, 오른쪽은 작성란으로 구성함
- 법적 안내 문구가 필요한 장은 하단 고정 박스로 반복 표시함

### 강의용 HTML PPT 실습본

- 11개 장을 11개 핵심 실습 슬라이드로 전환함
- 각 실습 슬라이드에는 입력 폼, 생성 버튼, 복사 버튼, 결과 미리보기 기능을 넣음
- 한 슬라이드에는 질문 1개 또는 실습 1개만 배치함

### 홈페이지용 엔딩노트 소개 페이지

- "WLC 2급 수료 후 바로 쓰는 실습형 엔딩노트"라는 메시지로 소개함
- 11개 장의 흐름을 카드형 목차로 보여줌
- 구매 문의, 단체 교육 문의, 강사용 자료 문의로 연결함

## 7. 제작 시 주의 사항

- 삶잇다협동조합 표기를 통일함
- 죽음, 상실, 이별 중심 표현보다 삶, 의미, 존엄, 가족 배려, 실천 중심 표현을 우선함
- 법적·의료적 효력이 있는 문서처럼 보이지 않도록 "예비 작성", "생각 정리", "가족 대화 자료"라는 표현을 사용함
- 한 질문에는 한 가지 생각만 쓰게 함
- 강사용 자료에는 감정 반응이 큰 수강생을 배려하는 진행 문장을 포함함
